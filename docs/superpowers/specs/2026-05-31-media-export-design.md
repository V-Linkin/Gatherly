# 媒体另存为功能设计文档

## 概述

内容详情页的图片和视频支持另存为到本地其他路径，提供单个右键导出和批量导出两种方式。

## 命名规则

```
{平台名}_{文件夹}_{作者}_{序号}_{日期}.{扩展名}
```

- **平台名**：自定义平台名（`customPlatform.name`）
- **文件夹**：有则填，无则跳过该段
- **作者**：原始作者名，特殊字符替换为 `_`
- **序号**：先图片后视频，按媒体列表从左到右顺序，从 1 开始
- **日期**：`yyyyMMdd` 格式（如 `20260531`）
- **扩展名**：保持原文件格式

**示例：**
- 有文件夹：`小红书_笔记_棠棠_1_20260531.jpg`
- 无文件夹：`小红书_棠棠_1_20260531.jpg`

## 触发方式

### 1. 右键菜单（单个导出）

在媒体图片/视频上右键，弹出「另存为」选项，弹出系统文件夹选择器，保存单个文件。

### 2. 工具栏按钮（批量导出）

详情页工具栏增加「导出」按钮，支持批量导出所有媒体。

### 批量导出逻辑

- 如果正文有图片 → 弹出勾选框（媒体区域 / 正文图片 / 全部）
- 如果正文无图片 → 直接导出媒体区域，不弹框
- 支持正文图片的平台：GitHub、知乎、豆瓣（豆瓣正文图片暂未支持，待后续实现）

## 文件覆盖处理

如果目标路径已有同名文件，自动在文件名后追加 `_1`、`_2` 等后缀避免覆盖。

## 技术实现

### 新增文件

- `Utilities/MediaExporter.swift` — 导出核心逻辑（命名生成、文件复制、防覆盖）
- `Views/Components/ExportPickerSheet.swift` — 批量导出的勾选弹窗

### 修改文件

- `Views/Item/ItemDetailView.swift` — 添加右键菜单、工具栏导出按钮
- `Utilities/FilePicker.swift` — 添加 `pickExportFolder()` 方法

## 边界情况

- 媒体文件不存在（`localPath` 为空）→ 跳过
- 作者/平台名为空 → 跳过该段
- 正文图片在本地缓存中（`bodyImageCache`）→ 从缓存读取保存
- 正文图片不在缓存中 → 从远程 URL 下载保存
